Welcome to some sweet ruby core and gem rdocs!

Why? Because some rdocs aren't on the web, and lots aren't in the hanna theme, which is much easier to read.

Core docs: <a href=doc/ruby-1.9.1-p129>1.9.1</a> <a href=doc/ruby-1.8.7-p174>1.8.7</a> <a href=doc/ruby-1.8.6-p369>1.8.6</a> <a href=http://webri.tigerops.org>RI</a>.

Rubygem rdocs for <a href=http://allgems.ruby-forum.com>all rubyforge gems</a>.

Note: local rdocs are now optional.  If you want significantly faster local gem installs, add the following line to the bottom of ~/.gemrc

<code>gem: --no-ri --no-rdoc</code>
